# vovstandard-deploy

Deployment target repository for the VOV ecosystem.

## Deployment requirements

- GitHub Pages enabled
- Root deployment
- Custom domain: `vovstandard.com`
- No source development in this repository
- No documentation beyond deployment requirements
- Receives automated deploy-ready static artifacts only

## Architecture

`VOV-ecosystem` is the single source of truth for development and deploys to:

- `vovverified-deploy`
- `vovstandard-deploy`
- `offgridtv-deploy`
